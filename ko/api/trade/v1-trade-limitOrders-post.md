# Limit order

지정한 조건으로 지정가 주문을 합니다.<br/>
쿼리 파라미터에서 조건을 설정할 수 있습니다. 파라미터 설명을 참고하십시오.

## Endpoint URI

    POST https://openapi.bitbox.me/v1/trade/limitOrders
