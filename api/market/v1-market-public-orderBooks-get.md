# Order book

Retrieves records for the given coin pair from the order book.<br/>

> **Note**<br/>
> Only `X-API-Key` in the header parameter is needed.

## Endpoint URI

    GET https://openapi.bitbox.me/v1/market/public/orderBooks?coinPair={coinPair}&depth={depth}
