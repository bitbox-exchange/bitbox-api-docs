# Order book

주문장에서 지정한 coin pair의 거래를 조회합니다.<br/>

> **Note**<br/>
> 이 API는 헤더 파라미터에 `X-API-Key`만 전달하면 됩니다.

## Endpoint URI

    GET https://openapi.bitbox.me/v1/market/public/orderBooks?coinPair={coinPair}&depth={depth}
