# Market coin pair policy

Shows the latest data on BITBOX's coin pairs<br/>

> **Note**<br/>
> Only `X-API-Key` in the header parameter is needed.

## Endpoint URI

    GET https://openapi.bitbox.me/v1/market/public/coins/pairPolicy

## Request parameters

None
